ما هو TLPM؟

TLPM يبدو أنه اختصار لـ "Track Links Performance Monitor" أو إضافة مشابهة لتتبع أداء SEO. هذه النقاط API تشير إلى إضافة WordPress مثبتة على موقعك تقوم بـ:

  • تتبع أداء الصفحات (top-performers, worst-performers)
  • مراقبة التحسينات (most-improved, least-improved)
  • تتبع الترتيب (near-promotion)
  • التنبيهات والإشعارات (alerts)

لماذا من إسرائيل؟

التفسير المحتمل: هذا البوت على الأرجح ينتمي لخدمة SEO أو أداة تحليل مرتبطة بإضافة TLPM. قد يكون:

  • خادم الإضافة يقوم بجمع بيانات من موقعك باستمرار
  • خدمة طرف ثالث تراقب أداء موقعك
  • بوت يحاول استخراج بيانات SEO من موقعك
  • نشاط مشبوه يستهدف نقاط API المكشوفة

  • استهلاك الموارد: ~13,000 طلب/يوم تستهلك CPU وذاكرة الخادم
  • عرض النطاق: استهلاك غير ضروري للباندويث

التوصيات والإجراءات المطلوبة

1. فحص الإضافات المثبتة

ابحث عن إضافة TLPM أو أي إضافة SEO مشابهة في WordPress. تحقق من إعداداتها أو احذفها إذا لم تكن بحاجة لها.

2. حظر IP الإسرائيلي

أضف IP 64.176.165.8 إلى قائمة الحظر في:
• .htaccess
• Cloudflare
• إضافة أمان WordPress

3. تعطيل REST API

قم بتعطيل أو تقييد الوصول لـ WordPress REST API للزوار غير المسجلين.

4. Rate Limiting

فعّل تحديد معدل الطلبات للحد من عدد الطلبات المسموحة من IP واحد.

5. حماية صفحة الدخول

استخدم:
• Two-Factor Authentication
• تغيير رابط wp-admin
• Limit Login Attempts

6. استخدام Cloudflare

فعّل Cloudflare مع قواعد حماية Bot Fight Mode لحظر البوتات الضارة تلقائياً.
