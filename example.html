🖼️ Base64 Image in HTML — Explained

An interactive example to understand how images can be embedded as text in HTML

1 Normal Image vs Base64 Image

There are two ways to show an image in HTML. Compare them:

✅ Normal Way (file reference)

<!-- The browser FETCHES the file from the server --> <img src="photo.jpg">

Browser sends a request → Server sends the image file → Browser displays it

🔤 Base64 Way (data embedded)

<!-- The image data IS right here in the HTML --> <img src="data:image/jpeg;base64,/9j/4A...">

No request needed — the image is already inside the HTML as text!

4 Summary

So what was all that text you saw?

That long string of characters in your Analytics textarea was your entire image converted to text. Every pixel, every color, every detail — encoded as letters and numbers.

When you put that text inside <img src="data:image/jpeg;base64,...">, the browser decodes it back into the original image and displays it.

It's like translating a book into Morse code: the content is the same, just represented differently. Base64 is "Morse code" for images.
